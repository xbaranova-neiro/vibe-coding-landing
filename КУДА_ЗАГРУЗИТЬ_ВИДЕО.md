# Куда загрузить видео для мастер-класса (День 1)

Сайт использует **только свой плеер** по прямой ссылке на MP4: без перемотки, только пауза и громкость. Rutube и другие встраиваемые плееры не используются.

Видео нужно загрузить на хостинг, который отдаёт **прямую ссылку на файл** (при открытии ссылки в браузере начинается воспроизведение или скачивание MP4).

---

## Вариант 1: Яндекс Облако (Object Storage)

1. Зарегистрируйтесь: [cloud.yandex.ru](https://cloud.yandex.ru)
2. Создайте каталог → **Object Storage** → **Создать бакет**
3. Имя бакета — любое (латиница). Публичный доступ: **Чтение объектов**
4. Загрузите файл MP4 в бакет
5. Откройте объект → **Скопировать ссылку** (она вида `https://storage.yandexcloud.ru/имя-бакета/day1.mp4`)
6. В проекте откройте **js/config.js** и вставьте:
   ```js
   window.WIPECODING_DAY1_VIDEO_MP4 = 'https://storage.yandexcloud.ru/ваш-бакет/day1.mp4';
   ```

Платно по объёму и трафику; для одного видео на 4 ГБ — обычно небольшая сумма.

---

## Вариант 2: Cloudflare R2

1. Зарегистрируйтесь: [dash.cloudflare.com](https://dash.cloudflare.com)
2. **R2 Object Storage** → **Create bucket**
3. Загрузите MP4 в бакет
4. Включите публичный доступ к бакету (Public access / R2.dev subdomain)
5. Скопируйте публичный URL файла
6. В **js/config.js**:
   ```js
   window.WIPECODING_DAY1_VIDEO_MP4 = 'https://ваш-адрес.r2.dev/day1.mp4';
   ```

Есть бесплатный объём; подходит для больших файлов.

---

## Вариант 3: Сжать видео и положить в репозиторий

Если получится ужать видео **до 100 МБ** (например, 720p, средний битрейт), файл можно положить в папку **video/** в репозитории. Тогда в конфиге укажите:

```js
window.WIPECODING_DAY1_VIDEO_MP4 = 'video/day1.mp4';
```

На GitHub Pages эта ссылка будет работать как `https://ваш-логин.github.io/vibe-coding-landing/video/day1.mp4`.

Сжать можно в [HandBrake](https://handbrake.fr/) или через FFmpeg (например, 720p, 1–2 Мбит/с).

---

## Что не подходит

- **Google Drive** — не отдаёт прямой поток для плеера, в браузере видео не воспроизводится
- **Rutube / YouTube** — встраиваемый плеер не даёт полностью отключить перемотку
- **Обычная загрузка в репо** — файлы больше 100 МБ GitHub не принимает

После того как укажете ссылку в **js/config.js**, обновите страницу — плеер подхватит новый URL.
